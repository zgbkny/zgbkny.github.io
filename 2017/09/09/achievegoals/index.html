实现目标的步骤

发表于 2017-09-08 | 分类于 成长

本文内容来自于：雷.达里奥 《原则》

1、设定目标

事实上，你想要什么都行，但你无法拥有一切。

要实现目标就要有优先次序，包括舍弃看起来还不错的选项。

不要混淆“目标”和“欲望”，这很重要。目标是你真的想实现的东西，而欲望是你想要但会阻碍你实现目标的东西。例如，假设目标是身体健康，欲望就是吃可口但不健康的食物。这种结果不利于你实现健康这一目标。但从结果来看，目标是好的，欲望是不好的。

避免基于自己主观意愿设定目标。

实现目标的每一步都要单独操作，绝不互相影响。目标不能仅根据表面上的可实现度而进行设定。一旦开始投身到实现目标的过程中，会需要大量的思考、反思，耗费大量的时间来最终确定实现目标的方案和具体任务，所以设定目标时，不要一开始就下结论说能不能实现。

这需要你坚信没什么是办不到的。即时当下不知道怎么操作也不要紧。只要刚开始坚定这个信念，按这个过程依次推进，实现目标后，就会获得真正的信心。如果你愿意，刚开始可以设定一些简单普通的目标，小有成就后就能增强信念，个人志向也开始日趋宏达起来。

实现目标不仅仅只是前进。

总的来说，要实现梦想，第一步是真的知道自己想要什么，不混淆目前和欲望，不因缺乏全面分析的假想障碍而把自己限制住了。

  • 你有多了解自己想要什么？
  • 你最重要的目标是什么？
  • 你擅于设定目标吗？

2、发现问题，对问题零容忍

大多数问题都蕴含着改善良机。

问题越棘手，带来的反应越大，要想成功，你得：1）发现问题；2）对问题零容忍

若不能发现问题，你就无法解决问题，也就不能前行实现目标。所以，让问题显现出来十分关键。

人们不能很好地发现自身问题，主要是源于缺少意愿、天赋或技能

  • 直面“残酷的现实”令人不悦，人们会下意识忽略，不去想
  • 思考棘手的问题，可能产生焦虑，阻碍进步
  • 人们常常更担心表面上是否看起来有问题，而不担心能否实现预期目标，因此就会比自身可能产生问题的错误或弱点。这种对自身错误的反感非常普遍，因为人们认为犯错是一种缺陷，而没有把犯错当做个人进化过程中的重要一环。
  • 有时人们仅仅是对问题缺乏洞察力罢了
  • 有人不会区分大问题和小问题。世上没有完美的事情，所以每个地方都可能发现数不完的问题，要不能从这一对问题中分辨出大问题，实际上是不能顺利发现问题的

发现问题时，保持聚精会神和逻辑思维。

面对问题时采取情绪化的做法，比如寻求同情或斥责他人，是毫无用处的。无论是什么原因，要想成功就要越过障碍。记住，你所体验的痛苦是“成长的痛苦”，会锻炼你的性格，经受住了考验就能获得回报。发现问题就像是搜寻迷阵里镶嵌的宝石，解开了迷阵，就能获得宝石，让人生变得更好。持续这样做，会加速你进化的过程。如果你的逻辑思维很好，那你应该很乐于寻找问题，因为发现问题会让你更接近目标。

  • 你擅于察觉问题吗？
  • 你对上述问题的自我评估有信心吗？
  • 如果你对自我评估很有信心，请论述自信的原因。
  • 问题要精准且具体化。

问题发现的越准确，越容易对问题进行诊断和提供有效的解决解决方案。例如，与其说“大家不喜欢我”，不如具体一点，说说谁不喜欢你，在什么情况下不喜欢你。

不要混淆问题和原因。

一旦发现了自身的问题，必须采取零容忍的态度。

人们不能很好地发现自身问题，主要是源于缺少意愿、天赋或技能：

  • 直面“残酷的现实”令人不悦，人们会下意识忽略，不去想
  • 思考棘手的问题，可能产生焦虑，阻碍进步
  • 人们常常更担心表面上是否看起来有问题，而不担心能否实现预期目标，因此就回避自身可能产生问题的错误或弱点。这种对自身错误的反感非常普遍，因为人们认为犯错是一种缺陷，而没有把犯错当做个人进化过程中的重要一环
  • 有时人们仅仅是对问题缺乏洞察力罢了
  • 有人不会区分大问题和小问题。世上没有完美的事情，所以每个地方都可能发现数不完的问题，要不能从这一堆问题中分辨出大问题，实际上是不能顺利发现问题的

发现问题时，保持聚精会神和逻辑思维。

3、诊断问题

比起直接跳到解决方案，关注问题的诊断与解决方案的设计会有效得多。

一定要冷静，保持清晰的逻辑。

要认识到，从自己的错误中吸取教训，以及从对结果有影响的人的错误中吸取教训，都对消费问题十分关键。

痛苦 + 反思 = 进步

尽管这可能与你期望的不一样，但这就是你必须接受和应对的现实。无法回避的现实是，想成功，就要找出所有主要问题的根本原因，人们犯的错和缺点有时就是根本原因。所以要想成功，一定要愿意观察自身和他人的行为，这些可能是产生问题的原因。

成功诊断问题所需的主要特质包括：逻辑、发现多重可能性的能力、愿意帮助他人克服追寻真理时的自我设障情绪。

诊断问题：

  • 诊断问题时，你敢不敢碰触敏感问题？
  • 你愿意探寻根本原因吗？比如去了解人们的真实情况？
  • 你善于扮演观察者的角色，并融入到根本原因的诊断中去吗？

4、设计方案

方案设计和写电影剧本一样，形象化地描述在规定时间内为实现目标需要做那些事。

有效的方案设计需要把问题想透，能形象化描述事情的来源起因和未来发展方向。关键是要能跟讲故事一样，形象化地描述整个过程，从你来自何处（或你做过什么），到你现在的情况，以及未来发展如何，一直到实现目标。整个方案要随着时间轴日益具体形象化，就像看一部记录了你过去、现在和未来的电影一样。

然后把设计方案写下来，保证自己能随时看到。方案设计包括谁要在什么时候做什么。具体任务清单是根据设计方案设置的，但任务和方案又不尽相同，方案设计将你的目标同具体任务关联起来。想要成功，在关注具体任务的同时，就一定不能忽略目标或方案。要记得不断反复前后查看。

在预期时间框架下，并不是所有的计划都能完成，关键是关注无法完成的任务，问自己这个结果是否可以接受。这时就需要集思广益了，同他人展开讨论至关重要。如果无法在规定时间内完成必须完成的任务，后果极有可能阻碍你实现最终目标，这时就得绞尽脑汁去想，怎样调整方案，使得任务能够得以完成，或者你也可以降低目标难度

5、完成任务

很多人大大低估了良好工作习惯的重要性。知道每天需要做什么并具备良好的纪律来保证这些工作的完成是至关重要的。

擅于完成此步骤的人能令人放心地实施方案，这样的人很自律，是积极行动派，不会因日常大量的繁琐工作影响任务的落实，他们以工作结果为导向：热爱推动自己冲刺前行，突破重点，实现目标。

找到解决方案，有缺点也不要紧

最重要的是，问你自己，阻碍你成功最大的缺点是什么。

人人都有缺点，成功人士与平庸之辈的区别在于他们懂得发现缺点并采取解决方案。

发现个人缺点很难主要有两点原因：

  • 大多数人都不去搜寻自身缺点，就是因为“自我设障”，社会大环境告诉他们，优缺点是不好的，所以会觉得发现缺点很痛苦。真相是：人人都有缺点，人们通过了解缺点，知道如何应对缺点而成长与成功，那我们的社会将变得更高效幸福。
  • 具备缺点就像缺少一种感官–如果你无法形象描述这个感官是怎样的，也就很难觉察你在这方面的缺失。

价值观→ 1) 目标 → 2) 问题 → 3) 诊断 → 4) 设计 → 5) 任务

设计与落实目标实现计划:

  • 人生如一场游戏，要克服阻碍前行的障碍
  • 玩这场游戏，多多联系，熟能生巧
  • 这场游戏由一系列会带来后果的选择组成
  • 你无法阻挡问题和选择扑面而来，所以最好还是学学怎么去应对他们吧
  • 你可以随心所欲做选择，不过最好要考虑后果
  • 问题的出现会带来痛苦，但也会提醒自己去寻找解决问题的办法。出现问题不能作为沮丧或不作为的理由，要这样想就太蠢了，也是没有意义的，消极应对扑面而来的问题是有害的。
  • 我们进化的速度不尽相同，这个速度取决于你自己
  • 这个过程是否进展顺利取决于你是否精准地把握方方面面，包括评估自身优势，思考应对策略。
